# Screenshots

This folder contains screenshots of the Simple Location Tracker App in action.

## Required Screenshots

### 1. permission_request.png
Shows the app requesting location permissions from the user when first launched.

### 2. map_location.png
Shows the user's current location displayed on the map with a marker.

### 3. location_update.png
Shows the marker updating as the user moves, demonstrating real-time tracking.

## How to Capture Screenshots

### On Android Device
1. Press **Power + Volume Down** simultaneously
2. Screenshots are saved to the device's Pictures/Screenshots folder

### On Android Emulator
1. Click the camera icon in the emulator toolbar
2. Or use the keyboard shortcut (varies by OS)

### Using Android Studio
1. Run the app on a device or emulator
2. Click the camera icon in the Logcat toolbar
3. Save the screenshot to this folder

## Tips for Good Screenshots
- Use a clean device with good battery and signal indicators
- Capture at different zoom levels to show map detail
- Show the permission dialog clearly
- Demonstrate location updates with visible coordinate changes
- Consider using the emulator's location simulation for consistent results
